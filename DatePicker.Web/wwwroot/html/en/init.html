初始化
在之前的例子中，datepicker.min.js 初始化页面总是发生在页面加载完成之后，但是当我们手动修改了页面的时候需要再次初始化 datepicker 。
案例一：通过 js 改变了 date-val 的值，需要把改变呈现到界面，则调用 datepicker() 方法。
 示例：
<script>
  $(function () {
   $("#txt1").attr("date-val", "2017-03-16 13:05:15").datepicker();
  });
</script>
<body>
  <input type="text" name="txt" id="txt1" class="datepicker" />
</body>
案例二：整个 input 框都是用js输出到界面的，则调用 datepicker() 方法初始化。
 示例：
<script>
  $(function () {
   document.body.innerHTML = "<input type=\"text\" class=\"datepicker\" />";
   datepicker();
  });
</script>
<body>
</body>

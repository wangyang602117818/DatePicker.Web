Introduction
1) Easy use
   just add a javascript reference, then add a attribute class="datepicker" to the input Tag, ok！
2) No css conflict
   use iframe internal，css is loaded in the iframe，your project's css style no conflict with it！
3) Date restriction
   you can add attribute date-start , date-end in the input Tag, this can give user a date-range to select.
4) Multi language Supported
   up to now, we support two languages (Chinese & English).
5) Multi skin Supported
   up to now, we support two skins (cyan & orange).

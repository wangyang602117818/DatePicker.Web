Skins
Specify different css path can change different skins
1）Orange
Sample:
<script src="/datepicker/datepicker.min.js" datepicker-template="/datepicker/datepicker-template.html" datepicker-css="/datepicker/datepicker-orange.min.css"></script>
 
1）Cyan
Sample:
<script src="/datepicker/datepicker.min.js" datepicker-template="/datepicker/datepicker-template.html" datepicker-css="/datepicker/datepicker-cyan.min.css"></script>

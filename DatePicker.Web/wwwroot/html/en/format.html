date-show-format & date-use-format
 Sometimes,There are some differences between show-format and use-format
 date-show-format: format of the value("yyyy-MM-dd","yyyy/MM/dd","yyyy.MM.dd","dd Month yyyy")
 date-use-format: format of the date-val("yyyy-MM-dd","yyyy/MM/dd","yyyy.MM.dd","dd Month yyyy")
 date-val: The real value which will be used in the code and saved to database, The format will be determined by date-show-format and date-use-format, default format is yyyy-MM-dd

1) Sample One: if set date-val only, both date-show-format and date-use-format will use this format
 <input class="datepicker" date-val="2017-05-02 12:05:06"/>

2) Sample Two: if set date-show-format only, date-use-format will use this format
 <input class="datepicker" date-show-format="dd Month yyyy hh:mm:ss"/>

3）Sample Three: if set date-use-format only, date-show-format will use this format
 <input class="datepicker" date-show-format="yyyy/MM/dd"/>

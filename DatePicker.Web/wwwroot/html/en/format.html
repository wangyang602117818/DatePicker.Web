show-format & use-format
 Sometimes,There are some differences between show-format and use-format
 date-show-format: Set the format that will display to the front-end pages(the format of the value)("yyyy-MM-dd hh:mm:ss","yyyy/MM/dd hh:mm:ss","dd Month yyyy hh:mm:ss")
 data-val: The real value, We will use it in the code or save to databse("yyyy-MM-dd hh:mm:ss")
1) Sample one
 <input class="datepicker" date-show-format="dd Month yyyy hh:mm:ss"/>

 <input class="datepicker" date-show-format="yyyy/MM/dd"/>

Enable time
1) Sample One
 Specify a initial value to the Input Tag，if datepicker detects time in the given value,it will enable time automatically
 Sample:
 <input class="datepicker" date-val="2017-03-06 12:00:00"/>
 Result：
  

2) Sample Two
 Set date-show-format
 Sample:
 <input class="datepicker" date-show-format="yyyy-MM-dd hh:mm:ss"/>
 Result:
  

3) Sample Three
 Set date-use-format
 Sample:
 <input class="datepicker" date-use-format="yyyy-MM-dd hh:mm:ss"/>

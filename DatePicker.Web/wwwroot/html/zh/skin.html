皮肤
通过指定 css 的路径可以使用不同的皮肤
1）橙色
示例：
<script src="/datepicker/datepicker.min.js" template="/datepicker/datepicker-template.html" css="/datepicker/datepicker-orange.min.css"></script>
 
1）青色
示例：
<script src="/datepicker/datepicker.min.js" template="/datepicker/datepicker-template.html" css="/datepicker/datepicker-cyan.min.css"></script>

文本框样式
 因为整个datepicker是一个iframe，而文本框的样式，不可能在iframe内部定义，所以这些样式代码需要自己添加项目中去
1）橘色的文本框
示例：
.datepicker {
    background: url(../images/date-orange.png) no-repeat;
    background-position: right;
    padding-right: 0;
    padding: 4px 8px;
    border: 1px solid #ccc;
    border-radius: 2px;
    color: #333;
    outline: none;
}
.datepicker:focus {
    border-color: #51A7E8;
    background-color: #fff;
    box-shadow: 0 0 5px rgba(103, 166, 217, 0.5);
}
